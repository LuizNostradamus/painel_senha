# Painel de Senhas 

Um sistema de fila de atendimento com painel de senhas desenvolvido em Python com interface gráfica Tkinter.

## Descrição

Este projeto implementa um sistema completo de gerenciamento de filas para atendimento, composto por três interfaces principais:

- **Interface do Cliente**: Para solicitar senhas
- **Painel de Administrador**: Para gerenciar a fila de senhas
- **Painel de Exibição**: Para mostrar a senha atual em atendimento

## Funcionalidades

**Geração de Senhas**: Senhas são geradas automaticamente no formato `A###` (ex: A001, A002, A003)

**Gerenciamento de Fila**: Visualização em tempo real da fila de atendimento

**Painel Digital**: Exibe a senha atual sendo atendida em letras grandes e visíveis

**Notificações**: Mensagens informativas para o usuário ao gerar ou tentar chamar senhas
